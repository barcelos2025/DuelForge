
# DuelForge Proto (v0.1)
Protótipo simples e jogável para validar o conceito: arena 1v1 (player vs IA) estilo cartas.
- Flutter + Flame
- Estrutura modular (MVVM com Provider)
- Cartas via JSON (pronto pra crescer)

## Rodar
1) Instale Flutter (stable)
2) Na pasta do projeto:
   - flutter pub get
   - flutter run

## Assets nomeados (referência)
- assets/images/DF_SPLASH_01.png  (splash)
- assets/images/DF_CARD_BACK_01.png (verso das cartas)

## Observação
As entidades ainda são renderizadas como formas (placeholder). O pipeline já está pronto pra trocar por sprites depois.
